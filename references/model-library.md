## Routing rules

- **Panel mode** (deliverable-bound runs): do not load these files into the main context. Each category agent reads its own file in its own context. See `panel-protocol.md`.
- **Solo mode** (no-deliverable runs): load the one or two category files the exchange calls for, by theme above. Loading all five at once defeats progressive disclosure.
- **Deep-bench lookups** from `behavior-catalog.md` resolve here: find the model in the lists above, load its category file.
