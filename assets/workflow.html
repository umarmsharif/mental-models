mental-models · agent skill

How it works: dump → infer → spar → commit

Describe a problem once. The skill infers the run's shape, stress-tests your thinking against ~60 mental models, and commits to a position. Two modes, one confirm gate, no questionnaire. The worked example below follows one run: Northwind's AI agent hit 200 conversations in week 3, and the team wants to double marketing spend.

Phase A · Intake
Phase 0 · Dump

Describe it once

You paste or describe the situation in one go: the thesis, the stakes, whatever framing you already have. Short is fine, rough is expected. The skill asks nothing here.

Phase 1 · Inferred read & confirm

It reads you, then confirms

The skill infers the audience, what you're producing, the decision at stake, and the frame — all from the dump, running its frame-check behaviours as internal lenses. It shows that read and asks you to confirm or correct it in a single gate. It infers, then confirms; it does not interrogate.

Phase B · Stress-test
Phase 2 · Panel sparring

Five specialists attack at once

Five sub-agents launch in parallel, one per model category. Each spars your thesis against every model it owns and scores it 0–5 on grip, originality, and story. The discipline is adversarial: most models score 0–2, and a catch only counts if it cites specifics from your thesis.

Example · the parallel fan-out
A · Thinking B · Systems C · Execution D · Strategy E · Personal
B returns Churn at grip 5: "200 is gross, not net — no repeat rate measured."
Solo mode (personal thinking, no deliverable): instead of the panel, the skill runs the diagnostic itself and spars you interactively — an escalation ladder that descends one layer deeper each round until it reaches the assumption you can't defend.

Phase 2.5 · Meta ranking

One judge ranks the catches

A meta agent composites every score (8×grip + 6×originality + 6×story), penalizes clichés (−10) and lenses used in recent runs (−15), dedupes overlapping catches, and keeps the top five load-bearing models plus the one bedrock assumption that, if false, kills the thesis.

Compounding (grip 3) took the cliché penalty and dropped to the bench. Bedrock: "week-3 volume predicts week-12 retention."

Phase 3 · Verdict gate

You make the call

The skill presents the bedrock, the five strongest attacks, and the model table. You adjudicate once. In solo mode this is where bedrock is named instead, after the sparring ladder bottoms out.

Phase D · Commit & output
Phase 3.5 · Narrative path

It takes a position

The skill commits to one recommended story and names the alternatives it rejected. This is the point of the whole run: a stand on which story the evidence supports.

Example · the recommended path
The story is not "we hit 200, let's scale." It is "we hit 200 — here's what we need to know before we commit." Load-bearing: Churn, Regression to the Mean, Falsifiability.
↓
Phase 4 · Output

The reasoning, written inline

The diagnostic (seven artifacts, every item tagged [known] / [inferred] / [uncertain]), the model selection table, and the recommended position — written straight into the conversation. Nothing is handed off. Panel runs append to the usage log, which feeds the next run's recency penalty so the same five models don't carry every analysis.

What's preserved

The in-agent "spar, not admire" discipline.

It's what makes the panel beat a single generalist pass: every agent attacks, none admire.

Why a gate, not zero

Audience, deliverable, and frame are inferred, then confirmed.

Inferring silently risks a wrong frame poisoning all five agents. The one gate is the cheap insurance.

Two modes

Panel for substantive calls, solo for personal thinking.

Same intake and frame check; solo swaps the panel for an interactive sparring ladder.
